# Student Task Manager (Java Edition)

## 项目简介
这是一个专为大学生设计的极简命令行任务管理工具，使用 Java 开发。它可以帮助同学快速记录作业、考试和日常琐事。

## 功能列表
- ✅ 快速添加任务（自动记录日期）
- 📋 查看所有待办事项
- ☕ 基于 Java，跨平台运行

## 如何运行
1. 确保安装了 JDK (Java Development Kit)
2. 下载代码到本地
3. 编译代码: `javac TaskManager.java`
4. 运行程序: `java TaskManager`
